USB Tethering Mode

This mode allows you to share the Internet and access the FlyOS system dashboard and resources to your computer via USB

Before launch this mode, make sure:
  • FlyOS framework is running
  • Device is connected to PC via USB

    • * Recommended download FlyOS Client, a bridge for PC to manage FlyOS
